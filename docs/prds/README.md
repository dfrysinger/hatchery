# Product Requirements Documents (PRDs)

This directory contains PRDs for features built by the AI Product Organization.

## Naming Convention

```
docs/prds/<feature-slug>.md
```

## Lifecycle

1. **Draft** — Judge synthesizes from council review
2. **Review** — Human reviews via PR
3. **Approved** — Merged to main
4. **Archived** — Feature shipped, PRD retained for reference
